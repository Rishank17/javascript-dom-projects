# 🚀 JavaScript DOM Mini Projects

A collection of interactive **JavaScript DOM Manipulation Projects** built using **HTML, CSS, and Vanilla JavaScript**. These projects helped me strengthen my understanding of DOM manipulation, event handling, animations, and interactive user interfaces.

---

## 📌 Projects Included

| No. | Project | Description |
|-----|---------|-------------|
| 🎨 | **Color Changer** | Change the background using random colors or custom color input. |
| 👥 | **User Directory & Live Search** | Search users dynamically by name or email in real time. |
| 🧮 | **Calculator** | Perform basic arithmetic operations through an interactive calculator. |
| ➕ | **Add Friend** | Toggle friendship status similar to a social media application. |
| ❤️ | **Instagram Like Animation** | Double-click an image to display a heart animation. |
| 🖱️ | **Custom Cursor** | Replace the default cursor with a custom cursor effect. |
| ✨ | **Trailing Cursor** | Smooth glowing cursor animation using `requestAnimationFrame()`. |
